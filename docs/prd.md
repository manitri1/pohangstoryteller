# PRD: 포항 스토리 텔러

---

## 📌 1. Detailed Product Description

**포항 스토리 텔러**는 여행자의 취향에 맞춘 맞춤형 스토리 기반 여행 경험을 제공하는 **웹 플랫폼**입니다.  
사용자는 포항의 다양한 스토리 코스를 탐험하고, 디지털 콘텐츠를 즐기며, QR 기반 스탬프 투어와 기록 앨범을 통해 여행의 기억을 남길 수 있습니다. 또한 DIY 기념품 제작과 커뮤니티 기능을 통해 여행 경험을 확장하고 공유할 수 있습니다.

**핵심 가치**:

- **스토리 중심**: 단순한 관광 정보가 아닌 서사와 경험을 기반으로 여행 설계
- **디지털 기록**: 사용자의 경험을 자동으로 정리하여 앨범 및 기념품으로 확장
- **참여와 공유**: 커뮤니티 및 SNS 확산 기능을 통해 사용자 참여 극대화

## 📌 3. Core Features and Specifications

### (1) 스토리 탐험

- 메인 화면에서 **사용자 취향 기반 추천 코스** 제공
- 코스 카테고리: 맛집탐방 / 역사여행 / 자연경관 / 골목산책
- 입력: 관심사 선택, 여행 기간, 동반자 유형
- 출력: 추천 코스 리스트 및 카드 뷰 제공

### (2) 스토리 코스 상세

- 코스별 소개, 주요 방문 지점, 예상 소요 시간, 지도 기반 경로
- 콘텐츠: 스토리 텍스트 + 방문지 이미지 + 경로 시각화

### (3) 디지털 콘텐츠

- 방문지별 **브랜드 필름(60–90초)**, **숏폼(15–30초)**, 텍스트 기반 스토리 제공
- 사용자가 직접 숏폼 제작 및 저장 가능

### (4) 경험 기록: 디지털 스탬프 투어

- 각 방문지 QR 코드 스캔 → 스탬프 획득
- 모든 스탬프 획득 시 앱 내 보상 제공 (쿠폰, 배지 등)

### (5) 나의 기록 앨범

- 스탬프, 사진, 숏폼 자동 정리
- AI 기반 앨범 자동 구성 (날짜/장소/테마별)

### (6) 기념품 소장 (DIY 굿즈 제작)

- 앨범 콘텐츠 → ‘포항4컷’, 롤링페이퍼, 포토북 제작
- 주문/결제 연동

### (7) 커뮤니티

- 숏폼/사진 공유 → 앱 내 커뮤니티 + 외부 SNS 공유 버튼 제공
- 좋아요, 댓글 기능

### (8) AI 기반 챗봇 & 일정 추천

- 입력: 여행 기간, 예산, 동반자 유형, 관심사
- 출력: 맞춤 일정, 맛집/카페/관광지 추천
- 인터페이스: 채팅형 UX

## 📌 4. Suggested Additional Features

1. **AR 체험 기능**: 특정 관광지에서 AR 콘텐츠(역사 인물, 스토리 복원) 제공
2. **실시간 인기 코스 랭킹**: 사용자 방문/공유 데이터 기반 인기 코스 추천
3. **다국어 지원**: 외국인 관광객을 위한 영어/중국어/일본어 번역 제공
4. **여행 체크리스트**: 준비물/예산 관리 기능
5. **위치 기반 알림**: 방문지 근처 도착 시 관련 스토리 푸시 알림

## 📌 5. User Persona and Scenarios

### 🎯 타겟 사용자

- **20대·30대 친구/연인 여행객**
- **가족 단위 여행객**

### 📖 시나리오

**시나리오 1: 연인 여행객**

- 입력: "2박 3일, 연인, 예산 30만 원, 자연경관+맛집"
- 출력: 커플 맞춤 코스 추천 (영일대 해수욕장 → 포항운하 → 맛집)
- 활동: 방문지 QR 스캔 → 스탬프 획득 → 숏폼 촬영 및 SNS 공유

**시나리오 2: 가족 여행객**

- 입력: "1일, 가족, 역사여행"
- 출력: 어린이와 부모 모두 즐길 수 있는 역사 스토리 코스 추천
- 활동: 앨범 자동 생성 + ‘포항4컷’ 기념품 제작
